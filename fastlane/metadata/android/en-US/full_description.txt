<p><i>SensorServer</i> Android app which let you stream real-time sensor data from your phone to Websocket clients. Clients, including web browsers and other applications, are able to receive streamed data through the WebSocket client API. Since this application functions as a Websocket Server, you will require a Websocket Client API to establish a connection with the application. With this app, you can effortlessly stream real-time sensor data from your phone to any application, allowing you to monitor the device's movement, environment, and position in real-time.</p>
